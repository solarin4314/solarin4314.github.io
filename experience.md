### Server Engineer → Data Engineer

커리어 초반에는 게임 서버 엔지니어로 시작해  
서버 기능 개발, 운영툴 구축, 배포 자동화를 경험했습니다.  
이 과정에서 **데이터가 생성되는 지점과 서비스 흐름**을 깊이 이해하게 되었고,  
이는 이후 데이터 파이프라인을 설계하는 데 중요한 기반이 되었습니다.

이후 데이터 엔지니어로 역할을 확장하며  
단순한 로그 수집을 넘어 **대규모 트래픽 환경에서도 안정적으로 동작하는  
엔드투엔드 데이터 파이프라인과 분석 시스템**을 설계·운영해왔습니다.

Kafka 기반 로그 수집, 목적별 저장소 분리,  
Athena를 활용한 서버리스 분석, 사전 집계 구조를 통해  
**확장성·안정성·비용 효율성**을 동시에 고려한 데이터 아키텍처를 구축했고,  
자동화·모니터링·정합성 관리까지 포함한 **운영 중심 데이터 엔지니어링**을 지향하고 있습니다.
