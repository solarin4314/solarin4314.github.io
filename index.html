Experience

7년 이상 서버 엔지니어 및 데이터 엔지니어로 근무하며, 실제 서비스 환경에서 안정적이고 확장 가능한 데이터 파이프라인과 분석 시스템을 설계·운영해왔습니다.

커리어 초반에는 게임 서버 엔지니어로 근무하며 Node.js 기반 서버 기능 개발, 운영툴 개발, 배포 자동화를 경험했습니다. 이 과정에서 데이터가 생성되는 지점을 이해하는 것이 이후 데이터 파이프라인 설계의 중요한 기반이 되었습니다.

이후 Kafka–Logstash–S3/Elasticsearch–Athena–MongoDB–MySQL 기반의 엔드투엔드 데이터 파이프라인을 직접 설계·구현·운영하며, 실시간 모니터링과 배치 분석 워크로드를 분리한 구조로 대규모 로그 트래픽 환경에서도 안정성을 확보했습니다.

단순한 데이터 처리에 그치지 않고 운영 안정성·비용 효율성·운영 편의성을 설계 기준으로 삼아 배치 자동화, 모니터링, 알림 체계를 구축했습니다. 또한 콘텐츠별 리셋 시간 차이로 발생하던 지표 왜곡 문제를 해결하여 지표 정합성을 100% 확보한 경험이 있습니다.

Metabase 기반 데이터 조회 환경을 구축하여 기획·운영팀이 개발팀의 도움 없이도 지표를 확인할 수 있도록 지원했고, 이를 통해 조직 전체의 데이터 활용성과 의사결정 속도를 개선했습니다.
